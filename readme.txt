== Description ==

Zotapay specialises in online payment processing solutions worldwide.
These include but are not limited to credit and debit cards, alternative payment methods and local payment solutions.


ZotaPay Payment plugin is easy to install fast and secure.

== Installation ==

Please note, this gateway requires WooCommerce 2.6 and above.

= Automatic installation =

Automatic installation is the easiest option as WordPress handles all operations itself and all you need is your web browser. To do an automatic install of the WooCommerce Zotapay plugin, log into your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type WooCommerce Zotapay Payment Gateway and click Search Plugins. Once you've located our plugin you can you can install it by clicking "Install Now", then "Activate" buttons.


= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your web server via FTP application. The WordPress codex contains [instructions on how to do this](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
